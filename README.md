## COMMANDS FOR CMDER:

``` 
cd into folder
```
```
$ source bin/activate
```
```
$ pip3 install -r requirements.txt
```
```
$ python3 app.py
```

Python version:
3.6.7 with python3 venv (e.g.  python3 -m venv safe-youtube-service/)
